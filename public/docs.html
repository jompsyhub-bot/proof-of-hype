API Docs

The public MVP base exposes enough backend surface to build in real time: campaigns, creator binding, ingestion, scoring, leaderboards, and payout export.

Run Locally

npm start

Open http://localhost:8787, then use the app to seed demo data or create campaigns.

Core Endpoints

GET /healthService health and version metadata.
GET /api/progressPublic build progress, milestone status, and live database counts.
POST /campaignsCreate a Solana meme coin campaign.
GET /campaignsList campaigns.
POST /demo/seedCreate/seed the $WAGMI demo campaign and run scoring.
POST /ingest/xIngest real X posts with X_BEARER_TOKEN.
POST /workers/scoreScore posts and rebuild payout rows.
GET /campaigns/:id/leaderboardRank scored posts by creator and risk signal.
GET /campaigns/:id/payouts.csvDownload SPL payout CSV.

Production Path

Move SQLite to Postgres/Supabase, add managed auth sessions, configure X app credentials, add Helius holder enrichment, and deploy the Node service behind HTTPS.
