API Docs

The production foundation exposes the core backend surface: campaigns, creator binding, ingestion, scoring, leaderboards, setup checks, and payout export.

Run Locally

npm start

Open http://localhost:8787, enter your admin token in the operator console, then create campaigns or seed sample data.

Core Endpoints

GET /healthService health and version metadata.
GET /api/progressPublic product status, milestone status, and live database counts.
GET /api/setupProduction readiness checks for admin, X, Solana, wallet strict mode, and persistence.
POST /campaignsCreate a Solana meme coin campaign.
GET /campaignsList campaigns.
POST /demo/seedAdmin-protected sample campaign seeding for launch rehearsals.
POST /ingest/xIngest real X posts with X_BEARER_TOKEN.
POST /workers/scoreScore posts and rebuild payout rows.
GET /campaigns/:id/leaderboardRank scored posts by creator and risk signal.
GET /campaigns/:id/payouts.csvDownload SPL payout CSV.

Production Path

Configure ADMIN_TOKEN, X app credentials, X bearer token, Helius or Solana RPC, and strict wallet signatures. Then move persistence to Postgres/Supabase or attach a persistent Render disk.
